Excel Table Scraper

Grabs DataBodyRange of all Excel tables in workbook (visible and hidden) and places the information in a worsheet at the end called "Values". Uses the longest header row of all Excel tables in the "Values" worksheet. Make sure to check alignment.

Workbook Consolidator

Cycles through all workbooks in file folder (you get to choose) and brings all worksheets into one workbook. Option to open certain extensions (.xls,.xlsm, etc.) or use passwords. Used in conjunction with the Excel Table Scraper to collect all values in a folder.

Great for any process where it is necessary to collect recommendations from many divisions/areas.

Sheet Selector - UserForm

Provides a list of all open workbooks. Once selection is complete, then it can be used in other macros. This allows you to use ActiveWorkbook after selection.

Very simple, but extremely useful!
